# TRI ANH EDU - ADMIN DASHBOARD DESIGN PROMPT

Continue the existing TRI ANH EDU design system.

DO NOT redesign the brand.

Reuse:

* Typography
* Color system
* Spacing
* Shadows
* Components
* Design language

from the existing TRI ANH EDU public website.

The admin panel must feel like part of the same product ecosystem.

---

# PRODUCT CONTEXT

TRI ANH EDU is an online education platform focused on:

* HSA
* TSA

The admin dashboard is used by:

* Admin
* Teacher

Roles:

## Admin

Full access.

## Teacher

Limited access.

Teachers cannot:

* Manage users
* Manage roles
* View system settings
